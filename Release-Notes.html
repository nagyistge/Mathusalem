Version 1.0a1

First private release

IMPORTANT INFORMATION: Incremental backups generated by versions of Mathusalem earlier than 1.2 may have incorrect information in the data files (.xml.gz) for some entries, and this can make Mathusalem fail when restoring the corresponding items. However, the archive files (.dmg, .zip, .cpio) are correct, so if you open them directly in the Finder, you can still extract all the items fine.
To put these pre-1.2 backups back into a fully correct state, you must delete all the incremental backups at the destination (i.e. all files with "Revision-00002" or bigger), while preserving the initial backup (i.e. "Revision-00001").
